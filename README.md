Economic Data Analysis and Visualization
This project focuses on analyzing economic data using data science techniques to uncover patterns, trends, and relationships across different sectors and time periods. The dataset consists of economic indicators such as current prices, constant prices, industries, and institutional sectors.
Initially, the dataset contained missing values and inconsistencies. Data preprocessing was performed to clean and structure the dataset. Missing values in numerical columns were handled using mean imputation, while categorical columns were filled using mode values. This ensured that the dataset became complete and suitable for further analysis.
Exploratory Data Analysis (EDA) was conducted to better understand the dataset. Various visualizations were used to extract insights:
Histograms were used to analyze the distribution of current and constant prices.
Box plots helped identify outliers in the dataset.
Scatter plots revealed relationships between current price and constant price.
Correlation heatmaps showed strong relationships between numerical variables.
Line plots were used to study trends in economic indicators over time.
Bar charts highlighted top industries contributing to economic output.
Pie charts illustrated sector-wise contributions, showing dominance of certain sectors (e.g., government sector).
The analysis revealed that current price and constant price are strongly positively correlated, indicating consistent economic patterns. The data also showed skewed distributions with significant outliers, which are common in economic datasets.
This project demonstrates how data preprocessing and visualization techniques can be applied to real-world economic data to generate meaningful insights.
